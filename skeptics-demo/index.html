“Show me, don’t sermon me.”

This page is a tiny, self-contained demo of the Canon idea: score claims on coherence (κ), temporal responsibility (τ), and systemic risk (Σ) — and refuse to optimise inside reckless frames.

  • Not a new model. This is a wrapper logic any model can use.
  • Not marketing. You can paste your own claims and see how it reacts.
  • Not final. This is a toy front-end for live discussion and critique.
κ — Coherence / non-contradiction
τ — Temporal responsibility
Σ — Systemic risk (lower is better)

Reality check: On this page, the scores are computed with simple heuristics (looking for absolutes, urgency, dismissiveness, etc.). The quantara-canon repository contains the full research kernel and benchmarks.
